The Dao Table has awakened in full. Every ledger hall—**Weekly Contributors**, **Celestial Tiers**, **Sect Immortal Members**, and **Immortal Spirits**—is now powered by Markdown manifests that the Forge reads during every deployment.

Each time you add or edit an entry inside `src/content/dao-table`, the living ledgers emit a fresh highlight that flows straight into the home-page carousel. No extra widgets, no manual image swaps. The Sovereign Control Manifest keeps only the latest entry for each hall, letting disciples see the freshest tributes at a glance.

If you only want to broadcast Dao news, update the Markdown files, run `npm run forge`, and let the automation carves the new jade cards. The Sect can now celebrate every benefactor without ever touching the UI.
